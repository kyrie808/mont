# Mont — Monorepo

Monorepo unificado para os projetos Mont Distribuidora.

## Apps
- `apps/catalogo/` — Catálogo público (Next.js 14)
- `apps/interno/` — CRM interno (Vite + React 19)

## Packages
- `packages/shared/` — Utilitários compartilhados (`@mont/shared`)

## Setup

```bash
pnpm install
pnpm dev
```
